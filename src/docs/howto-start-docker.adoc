= HOWTO: start service as docker container

== Cleaning previosly started container

[source,shell]
----
docker ps -a
docker kill pocket-square-present; docker rm pocket-square-present
----

== Start service

[source,shell]
----
docker pull pocketsquare/pocket-square-present:$TAG
docker run -dit --name pocket-square-present --link pocket-square-sort-shuffle --link pocket-square-users --link pocket-square-similar-by-text -e LOCALHOST=http://188.166.174.189:8080 -p 8080:5000 pocketsquare/pocket-square-present:$TAG
----
